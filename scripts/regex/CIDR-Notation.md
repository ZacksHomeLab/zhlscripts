# CIDR Notation RegEx
```
/^\b(?:(?:2(?:[0-4][0-9]|5[0-5])|[0-1]?[0-9]?[0-9])\.){3}(?:(?:2([0-4][0-9]|5[0-5])|[0-1]?[0-9]?[0-9]))\b(\/([0-9]|[1-2][0-9]|3[0-2]))?$
```

## Matching Examples
```
192.168.0.1
192.168.0.1/0
192.168.0.1/1
192.168.0.1/2
....
192.168.0.1/32
```

## Non-Matching Examples
```
192.168.0.1/
192.168.0.1/00
192.168.0.1/01
192.168.0.1/33
192.168.0.1/34
192.168.0.1/asd
```
