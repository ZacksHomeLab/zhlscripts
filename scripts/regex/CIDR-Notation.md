# CIDR Notation RegEx

## IPv4 CIDR
```
^\b(?:(?:2(?:[0-4][0-9]|5[0-5])|[0-1]?[0-9]?[0-9])\.){3}(?:(?:2([0-4][0-9]|5[0-5])|[0-1]?[0-9]?[0-9]))\b(\/([0-9]|[1-2][0-9]|3[0-2]))?$
```

HCL Compatible:

```
^\\b(?:(?:2(?:[0-4][0-9]|5[0-5])|[0-1]?[0-9]?[0-9])\\.){3}(?:(?:2([0-4][0-9]|5[0-5])|[0-1]?[0-9]?[0-9]))\\b(\\/([0-9]|[1-2][0-9]|3[0-2]))?$
```

### Matching Examples
```
192.168.0.1
192.168.0.1/0
192.168.0.1/1
192.168.0.1/2
....
192.168.0.1/32
```

### Non-Matching Examples
```
192.168.0.1/
192.168.0.1/00
192.168.0.1/01
192.168.0.1/33
192.168.0.1/34
192.168.0.1/asd
```

## IPv6 CIDR
```
^(?:[0-9a-fA-F]{1,4}:){7}[0-9a-fA-F]{1,4}\/(?:[0-9]|[1-9]\d|1[0-2][0-8])$
```

HCL Compatible

```
^(?:[0-9a-fA-F]{1,4}:){7}[0-9a-fA-F]{1,4}\\/(?:[0-9]|[1-9]\\d|1[0-2][0-8])$
```

### Matching Examples

```
2001:0db8:85a3:0000:0000:8a2e:0370:7334/64
2001:db8:85a3::8a2e:370:7334/128
fe80::1/128
```
